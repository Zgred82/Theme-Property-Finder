# Theme Property Finder for Blender

**Theme Property Finder** is a productivity tool for Blender artists and theme creators. It allows you to search for specific theme properties (like colors, sizes, or boolean toggles) across all Blender editors and synchronize them with a single click.

## üöÄ Key Features

* **Global Search:** Find any theme property hidden deep within Blender's API using a simple keyword search.
* **Intelligent List:** Use the "List" button to pick from a generated list of all available property names.
* **Batch Synchronization:** Update multiple editors at once. If you change a color in one editor, you can instantly sync it to the others.
* **Smart Filtering:** Results are grouped by editor type (3D View, Node Editor, etc.) for better clarity.
* **Selection Control:** Quickly select or deselect matching items to refine your batch updates.

## üõ† Installation (Standard Add-on)

1. Download the **`Theme_property_finder.py`** file from this repository.
2. Open Blender.
3. Go to **Edit > Preferences > Add-ons**.
4. Click the **Install...** button at the top of the window.
5. Select the downloaded `.py` file.
6. Find **"User Interface: Theme Property Finder"** in the list and check the box to enable it.

## üìñ How to Use

1. Navigate to **Preferences > Themes**.
2. The **Theme Property Finder** panel is located at the top of the Themes section.
3. Type a property name (e.g., "Grid" or "Background") or use the **List** button to search.
4. Adjust the value of a property in one editor group.
5. Check the boxes next to other editors you want to synchronize.
6. Click the **Sync (Paste icon)** button to propagate the value to all selected targets.
